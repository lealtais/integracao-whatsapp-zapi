# Integracao Supabase Z-API

Script para envio de mensagens via Z-API buscando contatos no Supabase.

## Requisitos
- Python 3.8+

## Setup da Tabela no Supabase

Para que o script funcione corretamente, crie a tabela no Supabase utilizando o seguinte comando SQL no **SQL Editor**:

```sql
CREATE TABLE pessoas (
    id bigint generated by default as identity primary key,
    nome text,
    numero text
);
```
*O campo `numero` deve conter o telefone no formato internacional completo (ex: 5511999999999).*

## Variaveis de Ambiente (.env)

Crie um arquivo `.env` na raiz do projeto (com base no arquivo `.env.example`) e preencha as chaves:
- `SUPABASE_URL`: Link do projeto Supabase (Project Settings -> API).
- `SUPABASE_KEY`: Token de acesso public/anon (Project Settings -> API).
- `ZAPI_INSTANCE`: ID da sua instancia Z-API.
- `ZAPI_TOKEN`: Token da sua instancia Z-API.
- `ZAPI_CLIENT_TOKEN`: Token de seguranca da sua conta Z-API.

## Como rodar

1. Instale as dependencias do projeto:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute o script principal:
   ```bash
   python main.py
   ```
